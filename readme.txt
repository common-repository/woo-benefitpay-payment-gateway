Get payment using your BenefitPay Merchant QR code on your website.

== Description ==
This plugin allow you to accept payments using BenefitPay Merchant QR code. User can place order on your site and can pay using BenefitPay. Simplest way to collect payment directly into your BenefitPay merchant account and gain user engagements with easy checkout.

== Installation ==
1. Upload benefitpay-gateway folder to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Goto woocommerce->settings->checkout->BenefitPay
4. Provide required information and enable BenefitPay gateway
5. Please upload your BenefitPay Merchant QR code image in your Wordpress image gallery and provide link in \"BenefitPay QR image URL\".
6. Collect payment using BenefitPay Merchant QR.
